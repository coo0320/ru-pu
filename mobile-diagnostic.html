思考ループの基本パターン診断

以下の各文について、どの程度あてはまるかを評価してください：

0=全くあてはまらない、1=ほとんどあてはまらない、2=ときどきあてはまる、3=よくあてはまる、4=ほぼいつもあてはまる

A. 過去反芻型

1. 過去の失敗や恥ずかしい出来事を繰り返し思い出してしまう
2. 「あのとき別の言い方をしていれば...」など、過去の会話を何度も分析する
3. 過去の選択について後悔の念にとらわれることが多い
小計A: 0

B. 未来不安型

1. これから起こりうる問題について過度に心配してしまう
2. 最悪のシナリオを想像して不安になることが多い
3. 「もし〜だったら」という考えが頭から離れない
小計B: 0

C. 完璧主義型

1. 「十分ではない」という感覚にとらわれることが多い
2. 小さなミスを大きな失敗のように感じてしまう
3. 自分の仕事やパフォーマンスについて高すぎる基準を持っている
小計C: 0

D. 人間関係懸念型

1. 他者が自分をどう思っているか気になって仕方がない
2. 会話の後、自分の言動を何度も振り返ることがある
3. 「嫌われているのではないか」と心配することが多い
小計D: 0

E. アイデンティティ混乱型

1. 「自分は何者なのか」について考え込むことがある
2. 自分の人生の意義や目的について悩むことが多い
3. 「このままでいいのだろうか」と自問自答することがある
小計E: 0
